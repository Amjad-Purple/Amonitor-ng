# Amonitor-ng
 Amonitor-ng network defense tool using Scapy and Python. Monitors WiFi for ARP spoofing, Deauth attacks, and TCP/TLS traffic. Features multithreaded packet sniffing, DNS/ARP analysis, and real-time threat alerts. CLI-based Purple Team utility for detecting malicious activities on local networks. Designed for cybersecurity education in Morocco.
